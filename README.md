# MyHub [![Build Status](https://travis-ci.org/Arcterus/MyHub.png)] #

A simple GitHub client designed for iOS 7.

## Dependencies ##
#### Build ####

* `xcodebuild`
* iOS SDK (at least v7.0)

#### Runtime ####

* iPhone 4 or better

## Build Instructions ##

```bash
$ ./bootstrap.sh
$ xcodebuild
```
